## 环境配置

在运行项目之前，请确保已配置以下环境：

### 1. Python 环境
- **版本要求**：Python 3.8 或更高版本
- **依赖库**：
  使用以下命令安装所需的 Python 依赖：
  ```bash
  pip install -r requirements.txt

### 2. Node.js 环境
- **版本要求**：Node.js 安装v18.x.x 版本(可以使用fnm管理node版本) 

## 启动方法
    -在KBQA目录下以管理员权限启动命令框
    -使用以下命令启动程序:
        python main.py
    -访问http://127.0.0.1:8000/
    -进入界面，体验各项功能
